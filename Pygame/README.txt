Readme.txt file for “basic shooter game uwu” created 1/16/2020
Authors: Aileen Sun and Elena Osipyan

This file is the readme file for our pygame. It explains how to play the game, along with general information on it.

Python 3 must be installed, along with the Pygame module, and the Wing IDE in order to run this program. If you need help installing Python, 
use the website https://www.python.org/downloads/. If you need help installing Pygame, see https://www.pygame.org/wiki/GettingStarted. Finally, 
to install Wing, visit https://wingware.com/downloads/wing-personal. Clicking on “culminating.py” within the “Culminating” folder should run the game.
 
There should be no errors, but if there are, please notify the creators at aileen.sun@student.tdsb.on.ca and elena.osipyan@student.tdsb.on.ca. We will work on fixing it.

FAQ's,
Who are the main characters? 
IB the Devil and Sans Undertale™.

Copyright information: Player 1 is a copyright-free character, while player 2, Sans, is taken from the video game Undertale. The soundtrack in the background 
are taken from the soundtrack of Pokemon Red/Blue. Specifically, the soundtracks taken were: Route 11, Victory! (Trainer Battle), Pokemon Gym, Vermillion City. 
Sound effects were downloaded from the website Zapsplat. All art were drawn by the creators.
